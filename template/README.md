# Mon harness

Un harness de skills multi-provider (Claude Code, OpenCode, Codex), basé sur le workflow engineering de Matt Pocock.

## Structure

- `skills/` — tes skills, au format Agent Skills (`<nom>/SKILL.md`). Édite-les, ajoute les tiens : les trois outils lisent le même format.

## Brancher le harness à tes outils

Le branchement (détection des outils installés et création des symlinks) arrive avec le script `setup.sh`.

## C'est à toi

Ce dossier t'appartient. Versionne-le dans git si tu veux et fais-le évoluer librement.
